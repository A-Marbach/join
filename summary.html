What is Join?

A task manager inspired by the Kanban method. You can create and organize tasks using drag and drop, assign tasks to users and define subtasks.

How to use it

1. create an account and add your contacts.

2. create as many tasks as you want.

3. On the board page you can edit your tasks and move them to the respective department.
